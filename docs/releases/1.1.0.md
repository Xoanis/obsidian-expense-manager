# Expense Manager 1.1.0

Release date:

- 2026-04-13

Highlights:

- finance email sync now uses the shared `obsidian-email-provider` plugin instead of built-in mailbox transport
- one sync run can work with one or many selected email-provider channels, and the saved cursor is tied to that exact channel set
- message hydration became more efficient: attachments are materialized lazily, only when the finance pipeline actually needs them
- rebuild compatibility was preserved for older finance notes that still carry legacy IMAP identity
- legacy built-in mail transport settings, including the old global mailbox-scope setting, are normalized away on load

Operational notes:

- verify channels and mailbox scope in `obsidian-email-provider` after upgrading from older built-in transport settings
- `obsidian-para-suite-contracts` dependency is aligned to `v0.2.0`
